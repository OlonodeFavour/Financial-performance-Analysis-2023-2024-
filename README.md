## 📌 Overview
This project presents a **Financial Dashboard (2023–2024)** developed using **Microsoft Excel** to analyze organizational financial performance.

The dashboard provides insights into key financial metrics such as income, expenses, vendor performance, departmental spending, and payment methods. It is designed to support financial decision-making through clear and interactive visualizations.

---

## ❗ Problem Statement
Organizations often face challenges in tracking and analyzing financial data effectively due to the complexity and volume of transactions.

Common issues include:
- Difficulty monitoring departmental expenses  
- Lack of visibility into major cost drivers  
- Poor tracking of vendor performance  
- Limited understanding of income vs expenses trends  
- Inefficient analysis of payment methods  

This project addresses these challenges by transforming financial data into a structured and interactive dashboard.

---

## 🛠️ Tools Used
- **Microsoft Excel**
  

---

## 🧹 Data Cleaning and Preparation
The dataset was cleaned and prepared to ensure accuracy and usability:

- Removed duplicate records  
- Handled missing values  
- Standardized financial categories (expenses, departments, vendors)  
- Ensured consistent formatting of currency values  
- Structured the dataset into tables for analysis  

---

## 🔍 Exploratory Data Analysis (EDA)
EDA was performed to uncover patterns and trends in financial data:

- Analyzed departmental financial performance  
- Identified top expense drivers  
- Evaluated vendor contributions  
- Examined transaction status distribution  
- Analyzed frequency of payment methods  
- Compared monthly income and expenses  

---

## 📈 Results and Findings
Key insights from the dashboard include:

- Certain departments account for the highest share of expenses  
- A few expense categories dominate total spending  
- Top vendors contribute significantly to overall transactions  
- Most transactions are successfully completed, with minimal failures  
- Digital payment methods are used more frequently than others  
- Monthly income and expenses fluctuate, indicating financial cycles  

---

## 💡 Recommendations
Based on the analysis, the following recommendations are suggested:

- Monitor and control high-spending departments  
- Reduce reliance on major expense drivers where possible  
- Strengthen relationships with top-performing vendors  
- Improve systems to minimize failed or pending transactions  
- Encourage efficient and cost-effective payment methods  
- Implement budgeting strategies based on monthly trends
